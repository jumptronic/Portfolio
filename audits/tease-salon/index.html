Needs attention

Fix it

No primary headline

The homepage has no H1 — the main heading search engines use to understand a page. Your logo fills that role visually, but it's an image, not text. Google can't read it the same way.

Add a text-based headline to the hero. It can be visually styled to match your brand — it doesn't have to look like a plain heading.

Fix it

Missing meta description

There's no custom description set for any page. When your site appears in Google search results, the snippet shown is auto-generated — usually awkward and not very convincing.

Write a one or two sentence description for each page in your site settings. Think of it as a mini ad for each page.

Fix it

Screenshot images used as site photos

Several images on the site are macOS screenshots. These are larger files than they need to be and load more slowly than proper web images.

Re-export as JPG or WebP and re-upload with descriptive filenames — e.g. "tease-salon-interior.jpg" rather than a timestamp.

Fix it

Images missing descriptive text

Most photos on the site have no alt text. This affects accessibility for visitors using screen readers, and means your images won't appear in Google image search results.

Add a brief description to each content photo — what's shown, where it is (e.g. "Color service at TEASE Salon, Saint Paul").

Fix it

Filename captions on services page

A gallery near the bottom of the services page is showing raw file names as visible captions: "1.png", "2.png", etc.

Remove the captions or replace with meaningful labels.

Fix it

No social sharing preview

When someone shares your site on Instagram, Facebook, or texts a link, the preview will look blank or generic — no image, no description.

Set a social sharing image (1200×630px) and short description in your site settings.

Keep an eye on

Watch it

Homepage title could be stronger

Your homepage tab reads "TEASE Salon" — accurate, but it misses a chance to tell search engines what you do and where.

Consider: "TEASE Salon | Hair Color & Cuts in Saint Paul, MN"

Watch it

Cancellation policy as first impression

The policy is detailed and necessary — but it's one of the first things a new visitor reads. For someone just discovering TEASE, it can feel like rules before a welcome.

Consider a brief, warm note on the homepage and move the full policy to the booking flow.

Watch it

Logo appears twice in navigation

The logo loads twice per page — once for desktop nav, once for mobile. Both have correct alt text. Just something to know if page load speed becomes a concern.

Watch it

Inconsistent page title format

The homepage uses "TEASE Salon" while interior pages use "Page Name — TEASE Salon." Small thing, but consistency is a tidy SEO habit.

Standardize to one format across all pages.

Watch it

Image load performance (unconfirmed)

Worth running a quick speed test to confirm images — especially the video backgrounds — aren't slowing things down on mobile.

Run a free check at pagespeed.web.dev.
